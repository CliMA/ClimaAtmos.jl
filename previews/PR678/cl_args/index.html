│ t_end                     │ String  │ Simulation end time. Examples: [`1200days`, `40secs`]                                                                                                       │
│ dt                        │ String  │ Simulation time step. Examples: [`10secs`, `1hours`]                                                                                                        │
│ dt_save_to_sol            │ String  │ Time between saving solution. Examples: [`10days`, `1hours`, `Inf` (do not save)]                                                                           │
│ dt_save_to_disk           │ String  │ Time between saving to disk. Examples: [`10secs`, `1hours`, `Inf` (do not save)]                                                                            │
│ config                    │ String  │ Spatial configuration [`sphere` (default), `column`]                                                                                                        │
│ moist                     │ String  │ Moisture model [`dry` (default), `equil`, `non_equil`]                                                                                                      │
│ microphy                  │ String  │ Microphysics model [`nothing` (default), `0M`]                                                                                                              │
│ forcing                   │ String  │ Forcing [`nothing` (default), `held_suarez`]                                                                                                                │
│ vert_diff                 │ Bool    │ Vertical diffusion [`false` (default), `true`]                                                                                                              │
│ coupled                   │ Bool    │ Coupled simulation [`false` (default), `true`]                                                                                                              │
│ turbconv                  │ String  │ Turbulence convection scheme [`nothing` (default), `edmf`]                                                                                                  │
│ turbconv_case             │ String  │ The case run by Turbulence convection scheme [`Bomex` (default), `Bomex`, `DYCOMS_RF01`, `TRMM_LBA`, `GABLS`]                                               │
│ anelastic_dycore          │ Bool    │ false enables defualt remaining tendency which produces a compressible model, the true option allow the EDMF to use an anelastic dycore (temporary)         │
│ hyperdiff                 │ Bool    │ Hyperdiffusion [`true` (default), `false`]                                                                                                                  │
│ idealized_insolation      │ Bool    │ Use idealized insolation in radiation model [`false`, `true` (default)]                                                                                     │
│ idealized_h2o             │ Bool    │ Use idealized H2O in radiation model [`false` (default), `true`]                                                                                            │
│ idealized_clouds          │ Bool    │ Use idealized clouds in radiation model [`false` (default), `true`]                                                                                         │
│ rad                       │ String  │ Radiation model [`nothing` (default), `gray`, `clearsky`, `allsky`, `allskywithclear`]                                                                      │
│ energy_name               │ String  │ Energy variable name [`rhoe` (default), `rhoe_int` , `rhotheta`]                                                                                            │
│ upwinding                 │ String  │ Upwinding mode [`none` (default), `first_order` , `third_order`]                                                                                            │
│ ode_algo                  │ String  │ ODE algorithm [`Rosenbrock23` (default), `Euler`]                                                                                                           │
│ split_ode                 │ Bool    │ Use split of ODE problem. Examples: [`true` (implicit, default), `false` (explicit)]                                                                        │
│ regression_test           │ Bool    │ (Bool) perform regression test                                                                                                                              │
│ enable_threading          │ Bool    │ Enable multi-threading. Note: Julia must be launched with (e.g.,) `--threads=8`                                                                             │
│ output_dir                │ String  │ Output directory                                                                                                                                            │
│ job_id                    │ String  │ Uniquely identifying string for a particular job                                                                                                            │
│ trunc_stack_traces        │ Bool    │ Set to `true` to truncate printing of ClimaCore `Field`s                                                                                                    │
│ fps                       │ Int64   │ Frames per second for animations                                                                                                                            │
│ post_process              │ Bool    │ Post process [`true` (default), `false`]                                                                                                                    │
│ h_elem                    │ Int64   │ number of elements per edge on a cubed sphere                                                                                                               │
│ z_elem                    │ Int64   │ number of vertical elements                                                                                                                                 │
│ z_max                     │ Float64 │ Model top height. Default: 30km                                                                                                                             │
│ z_stretch                 │ Bool    │ Stretch grid in z-direction. [`true` (default), `false`]                                                                                                    │
│ dz_bottom                 │ Float64 │ Model bottom grid depth. Default: 500m                                                                                                                      │
│ dz_top                    │ Float64 │ Model top grid depth. Default: 5000m                                                                                                                        │
│ kappa_4                   │ Float64 │ Hyperdiffusion parameter                                                                                                                                    │
│ rayleigh_sponge           │ Bool    │ Rayleigh sponge [`true`, `false` (default)]                                                                                                                 │
│ viscous_sponge            │ Bool    │ Viscous sponge [`true`, `false` (default)]                                                                                                                  │
│ zd_rayleigh               │ Float64 │ Rayleigh sponge height                                                                                                                                      │
│ zd_viscous                │ Float64 │ Viscous sponge height                                                                                                                                       │
│ kappa_2_sponge            │ Float64 │ Viscous sponge coefficient                                                                                                                                  │
│ apply_moisture_filter     │ Bool    │ Apply filter to moisture                                                                                                                                    │
│ disable_qt_hyperdiffusion │ Bool    │ Disable the hyperdiffusion of specific humidity [`true`, `false` (default)] (TODO: reconcile this with ρe_tot or remove if instability fixed with limiters) │
│ start_date                │ String  │ Start date of the simulation                                                                                                                                │
